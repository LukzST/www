Installation Guide

System Requirements

Toaster Mode (Minimum)

  • OS: Windows 10 (or a 2015 toaster)
  • Processor: Any dual-core CPU (a hamster on a wheel also works)
  • Memory: 1 GB RAM (your smartwatch has more)
  • Storage: 500 MB (about 50 toasts worth of data)
  • Graphics: A potato with copper wires

Smart Toaster Mode (Recommended)

  • OS: Windows 11 (or an air fryer with WiFi)
  • Processor: Any quad-core CPU (a very motivated squirrel)
  • Memory: 2 GB RAM (two potatoes in parallel)
  • Storage: 1 GB available space
  • Graphics: A sweet potato (premium version)
  • Note: Game may toast your bread while you play. Not responsible for burnt toast.

How to Install

1

Download the game

Download LIGHT from Itch.io or GitHub.

2

Extract the files

Extract the downloaded ZIP file to a folder of your choice.

3

Run the game

Double-click LIGHT.exe to start playing.

Terminal alternative 
# Clone the repository
git clone https://github.com/LukzST/LIGHT.git

# Navigate to the directory
cd LIGHT

# Run the game (if compiled)
./LIGHT.exe

Gameplay Guide

Core Mechanics

Energy Management

Complete protocol sequences in order. Failure resets progress and drains energy.

The Fade & Survey

Your answers determine if you access the mainframe. Wrong answers lock you out.

Core Stabilization

Dynamic minigame with 3 modes: BAR SYNC, PULSE SYNC, and KEY HACK.

PACPRO Subsystem

PAC-MAN style minigame. Complete all 3 levels to unlock ELITE OPERATOR status.

Terminal Commands

Use help, status, logs, scan, repair, clear, exit to interact with the system.

Save System

Automatic saves at key story moments. Use CONTINUE MISSION to resume.

Tips & Tricks

  • Complete protocols in order - Wrong sequence resets progress.
  • Type "6789" fast - You have 5 seconds during lockdown.
  • Use "lux1999files" - Decrypt the Project Fade logs.
  • Press [S] during data leaks - Export files to your desktop.
  • Use "LUX4LIFE" - Alternative password for secret route.

Troubleshooting

Game crashes on startup ▼

Make sure your system meets the minimum requirements. Try running as administrator or reinstalling the game.

Black screen on launch ▼

Update your graphics drivers. If the issue persists, try launching in windowed mode via config file.

Audio not working ▼

Check your system volume and ensure no other application is using exclusive audio control.

Save files not loading ▼

Check that the save directory has write permissions. Save files are located in /CONFIG/checkpoints.json

How to report a bug? ▼

Report bugs on GitHub Issues with screenshots and steps to reproduce.
